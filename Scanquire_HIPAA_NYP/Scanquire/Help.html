About

A plugin based scanning solution.

Hotkeys

* These hotkeys require the use of a number pad.
Control Panel
[F1]Show Help
[Ctrl][I]Insert From Scanner
[Ctrl][N]New From Scanner
[Ctrl][O]New From Archive
[Ctrl][G]Append From Scanner
[Ctrl][S]Send To Archive
Page Navigation
[Home]Go To First Page
[End]Go To Last Page
[Page Up]Go To Previous Screen
[Page Down]Go To Next Screen
[Left Arrow]Go To Previous Page
[Right Arrow]Go To Next Page
[Up Arrow]Go To Previous Line
[Down Arrow]Go To Next Line
Thumbnails And Page Processing
[Ctrl][0]*Fit Window
[Ctrl][1]*Actual Size
[Ctrl][2]*Fit Width
[Ctrl][3]*Fit Height
[Ctrl][+]*Zoom In
[Ctrl][-]*Zoom Out
[Ctrl][A]Select All Pages
[Ctrl][T]Invert Check Selected Items
[Ctrl][Y]Check Selected Items
[Ctrl][U]UnCheck Selected Items
[Ctrl][R]Rotate Selected Pages Right 90 Degrees
[Ctrl][E]Rotate Selected Pages Left 90 Degrees
